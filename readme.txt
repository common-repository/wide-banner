== Description ==

EN
Display one (or more) banner on your website. Easy, simple and free banner plugin for wordpress.
You can customize the color, the text, the button and the link

FR
Créez facilement une ou plusieurs bannière(s) sur votre site web. Personnalisez le texte, la couleur et le lien de votre bannière.


== Installation ==

= From your WordPress dashboard =
EN
1. Visit 'Plugins > Add New'
2. Search for 'Wide Banner'
3. Activate 'Wide Banner' from your Plugins page.
4. Visit 'wide Banner' in the sidebar to create a new banner.
5. Activate/Desactivate banners in 'Wide Banner>list of banners'

FR
1. Allez à la page des Plugins et cliquez sur "ajouter"
2. Recherchez 'Wide Banner'
3. Activez 'Wide Banner' a partir de votre page de plugins
4. Dans le menu de gauche cliquez sur 'wide Banner' pour créer une bannière
5. Activez ou désactivez vos bannières à la page 'Wide Banner>list of banners'

= From WordPress.org =
EN
1. Download 'Wide Banner'.
2. Upload the 'wide-banner' directory to your '/wp-content/plugins/' directory
3. Activate 'Wide Banner' from your Plugins page.
4. Visit 'wide Banner' in the sidebar to create a new banner.
5. Activate/Desactivate banners in 'Wide Banner>list of banners'

FR
1. Téléchargez 'Wide Banner'.
2. déplacez le repertoire 'wide-banner' dans le dossier '/wp-content/plugins/'
3. Activez 'Wide Banner' à partir de votre page de plugins
4. Dans le menu de gauche cliquez sur 'wide Banner' pour créer une bannière
5. Activez ou désactivez vos bannières à la page 'Wide Banner>list of banners'
